# Katrina's Journal 2026

A personalized digital journal and planner web app designed as a heartfelt gift for tracking moods, thoughts, and daily plans throughout 2026.

## About This Project

This project was created as a special gift for my girlfriend, Katrina, who wanted a digital planner with mood tracking capabilities. She designed the entire layout and aesthetic in Canva, featuring her preferred color palette, playful Filipino phrases, and a cozy handwritten style. I then brought her vision to life by coding it into a fully functional web application.

**Current Status**: This is an ongoing project! I'll be continuously updating and improving features based on feedback and usage.

## Important Notes

- **Privacy First**: All journal entries and data are stored locally in your browser using localStorage. Nothing is ever sent to any server or database.
- **Device-Specific**: Data saved on one device won't automatically appear on another device. Each browser/device maintains its own separate journal.
- **Browser Data**: Clearing your browser's cache/data will delete all journal entries. Avoid clearing site data if you want to keep your journals.
